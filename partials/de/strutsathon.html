Hackathon

Wir hacken, ganztägig.

Nach ein paar einleitenden Worten entscheiden wir, wer an welchen Features arbeitet. Gruppenbildung ist gewünscht, und Zusammenarbeit ohnehin. Die Struts-Committer werden eine Menge Ideen mitnehmen, die unter Umständen grundlegend für Struts 3 sein werden.

Jeder ist Willkommen, solange er Java spricht. Ihr müsst keine Struts-Experten sein, im Gegenteil. Am Ende des Tages möchten wir nicht nur großartigen Code aufweisen. Wir möchten es Struts-Neulingen ermöglichen, in das Framework einzusteigen.

Wir werden an diesem Tag so arbeiten, wie wir es eben bei Apache so tun. Es geht also nicht nur um einen Einstieg in Struts, es geht auch um einen Einstieg in die Apache Software Foundation. Wir möchten Lust auf Open Source wecken und zeigen, dass die ASF kein Buch mit sieben Siegeln ist.

Kosten + Teilnahmebedingungen

Die Kosten stehen noch nicht fest - wir hoffen, Sponsoren zu finden und das Event kostenlos durchführen zu können. Ansonsten wird der UKB bei etwa 30 € liegen. Stay tuned!

Um eueren Code wirklich in's Struts Repository committen zu können, benötigen wir eure schriftliche Erlaubnis. Bitte sendet dazu folgendes Formular (ICLA) an die Apache Software Foundation (Adresse ist dort angegeben).

Formular: ICLA

Ohne ICLA ist keine Teilnahme möglich. Bitte sendet das ICLA spätestens eine Woche vor Teilnahme an die ASF. Falls es Probleme oder Fragen dazu gibt, stehen wir euch gerne zur Verfügung.
