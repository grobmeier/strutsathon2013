Hackathon

We are hacking the whole day.

After a short introduction we will decide what things we want to develop. Most likely we make up teams and cooperate. The Struts-committers will bring a lot of ideas for the Hackathon. Some of them will maybe be the base for Struts 3.

Everybody who speaks Java is welcome. You don't need to be a Struts-expert. At the end of the day we have some great code at hands. But we also want to make it easy for Struts newbiews to join into the framework.

At this day we work as committers work at Apache. That said it is not only a first step into Struts, it is also a first step into the Apache Software Foundation. We would like you to know how much fun Open Source is. We would like to demonstrate the Apache Software Foundation is an open and welcoming organization.

Donate code

If you would like to see your code in the Struts repository we do need your permission. This is done by sending an ICLA to the Apache Software Foundation. Please see instructions in the ICLA itself:

ICLA

You can join the Hackathon without ICLA, but we can't use your code then. If you would like to experiment a little with Struts, it's OK. But if you would like to work on the Struts-Core, it's better to submit an ICLA.

If you have questions on the ICLA, please contact us.
